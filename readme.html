1) Erstellen Sie ein Formular, in welchem ein Nutzer Daten eingeben kann. (5 P)

Das Formular soll folgende Elemente beinhalten

  • 2 Textfelder

    • Gib deinen Benutzernamen ein (mind. 5 Buchstaben):

    Gib deine E-Mail-Adresse ein (Pflichtfeld): Hier wurde als type nicht "email" genommen, da "text" gefordert war.

  • Drop-Down-Menü mit mind. 6 Optionen

    • Wähle Schwierigkeitsstufe(max. 2 Angaben): 6 Schwierigkeitsstufen

  • Radiobuttons mit mind. 3 Optionen

    • Wähle eine Kategorie aus: mit Kategorien

  • 1 Feld zur Eingabe von Zahlen

    • Wähle bis zu 8 Fragen:

  • Eine Datalist mit mind. 3 Einträgen

    • Wähle einen Bootstyp aus:

2. Implementieren Sie Vorgaben für die Benutzereingaben (4 P)

Soweit möglich, verwenden Sie für die Bedingungen HTML-Attribute, andernfalls implementieren Sie Prüfungen mit JavaScript unter Aufgabe 3.

  • Textfeld 1 muss mindestens 5 Buchstaben lang sein

    • über Attribut: minlength="5", sowie über die Funktion check() geprüft

  • Textfeld 2 ist Pflichtfeld

    • über Attribut: required, sowie über checkValidity()

  • Es muss mindestens ein Radiobutton ODER eine Dropdown-Option ausgewählt sein

    • Über die Funktion check()

  • In der Drop-down-Liste dürfen nur max. 2 Einträge markiert werden.

    • Über die Funktion check()

  • Die eingegebene Zahl muss zwischen 1 und 8 liegen

    • über Attribut: min="1" und max="8" und über die Funktion check()

3. Nehmen Sie eine Formularvalidierung vor (20 P)

Markieren Sie die Textfelder mit gelbem Hintergrund bei fehlerhafter Eingabe.

Erstellen Sie eigene Fehlermeldungen für fehlerhafte Eingaben bei allen eigeneten Formularfeldern.

Prüfen Sie per JavaScript, ob die unter b) genannten Bedingungen erfüllt wurden, bevor das Formular abgesendet wird. Benutzen Sie für die Aufgabe checkValidity(), setCustomValidity() und das validityState-Objekt.

4. Geben Sie die Formulareingaben aus (15 P)

Speichern Sie die Formulardaten beim Absenden in ein JavaScript-Objekt.

Serialisieren Sie das Objekt in JSON.

Geben Sie die Formulardaten auf der HTML-Seite aus, indem Sie das JSON-Objekt in ein JavaScript-Objekt zurückwandeln und die Key-Value-Paare unterhalb des Formulars ausgeben. Nehmen Sie entsprechende Änderungen am DOM vor.

Geben Sie auch zur Überprüfung den Inhalt des JSON-Objekts aus.
